API

/data/:type

returns Distributor Detail by Distributor Type
JSON Structure
{ "accountId": AccountID, "Name": Account Name, "Type": Distributor Type, "Phone": Contact Number, "Website": Distributor website }

/opportunity/:AxCustomerId

returns Opportunity by AX Account Number
JSON Structure
{ "name": Account Name, "probability": Probability%, "amount": Amount }

/opportunity/:AxCustomerId

returns data from Salesforce, AX365 and AX2009 by AX Account Number
JSON Structure
salesforce : { "name": Account Name, "probability": Probability%, "amount": Amount }, AX365 : { customer: { "custAccount": Customer Account Number "Customer" : Name }, data: { "SalesID":sales ID, "createdDate" : Booked Date, "Amount" : amount } }, AX2009 : { customer: { "custAccount": Customer Account Number "Customer" : Name }, data: { "SalesID":sales ID, "createdDate" : Booked Date, "Amount" : amount } }
